Welcome to MOAFS's documentation!
=================================

``MOAFS`` is a library for the Massive Online Analysis (MOA) framework. It is based on the MOAReduction extension and contains seven 
feature selection algorithms to be used as dimensionality reduction techniques in data streams classification problems, especially in the text-domain field.
